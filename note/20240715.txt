JQuery
- 자바스크립트로 만들어진 라이브러리
- 자바스크립트의 DOM(Document Object Model)작업을 좀 더 쉽게 할 수 있는 기능을 갖고 있다.

JQuery 변수
- 변수명 앞에 $기호가 붙는다

var $변수명

let $변수명

const $변수명

JQuery 내장함수를 사용할 수 있다.

JQuery로 태그를 선택하는 법
- JQuery 선택자는 이름, ID, 클래스, 유형, 속성, 속성 값 등을 기반으로 HTML 요소를 찾는다.
- JQuety의 모든 선택자는 $와 ()로( $(선택자))로 구성된다

태그로 선택
$("태그명") -> document.getElementByTagName("태그명")

id로 선택
$("#아이디값") -> document.getElementById("아이디값")

클래스로 선택
$(".클래스값")

속성으로 선택
$("태그명[속성='속성값']")
