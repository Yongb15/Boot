product 조회

그 후 조회된 행들을 arrayList<>로 묶어야한다 그걸 mapper로 돌려줌

/register -> product-insert -> Controller -> Mapper -> xml -> Mapper -> Controller -> productMapper.insert(productVO);로 돌아오고 return new RedirectView("list"); 실행
